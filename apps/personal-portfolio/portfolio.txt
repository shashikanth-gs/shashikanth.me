🤖 AI & Agentic Platform Projects
AIOps — Intelligent Reactive Operations Platform

A production-grade, multi-agent AIOps solution built to transform how operations teams detect, diagnose, and respond to system incidents. The platform performs reactive root cause analysis across distributed systems and recommends — or autonomously executes — remediation actions based on confidence and policy.
Built on a foundation of custom SDK development layered on top of industry-leading agent frameworks, with protocol-level A2A integrations to compose a coherent, extensible agent mesh. Each deployment integrates with a pluggable, project-specific knowledge base that agents learn from and improve over time.
Integrates natively with: Splunk Prometheus/Thanos Grafana Tempo ServiceNow Kubernetes — and is architected to be fully extensible to new tooling and channels. Supports multiple channel integrations for alert ingestion and response delivery.


Autonomous Vulnerability Remediation Platform

An end-to-end AI-driven platform targeting the maintenance phase of the SDLC — specifically built to eliminate the manual toil of vulnerability management at scale. The platform autonomously identifies, fixes, validates, and raises PRs for the full spectrum of software vulnerabilities, including:

Base image upgrades
BlackDuck dependency vulnerabilities
SonarQube code quality issues
Fortify security findings
Framework version upgrades (e.g. Spring Boot, Java runtime upgrades)

Entirely language-agnostic — works across any tech stack. The pipeline runs e2e automation including test execution, with human-in-the-loop reserved only for PR review — keeping engineers in control without slowing the process. Integrates with JIRA for traceability and built end-to-end on the BMAD methodology, fully compliant with organizational security policies.


AI API Recommender System

An intelligent platform that semantically understands API capabilities, dynamically composes and recommends APIs to satisfy complex business requirements — powered by agentic AI, vector search, and multi-agent orchestration.

Event Intelligence Platform

A companion to the API Recommender — applying the same intelligence layer to event-driven ecosystems. Enables semantic discovery, routing, and orchestration of events across distributed airline platforms.

AI Knowledge & Specification Platform

A natural-language interface for navigating, querying, and reasoning over large API and specification landscapes — enabling developers and architects to explore complex API ecosystems without reading raw specs.
